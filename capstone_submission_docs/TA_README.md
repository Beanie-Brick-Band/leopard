# How to get started
Go to https://tryleopard.dev/. Sign in using the accounts provided. We recommend trying the teacher view first to set up whatever classrooms you want before trying the student accounts. You can also use incognito windows to have different accounts open at once.

We recommend you view `Group 18 - Capstone Final Video.mp4` for a demo to get a visual tour of our site before trying it yourself.  

# Teacher view
As a teacher, you can create and manage classrooms. In the classroom page, you can create and manage assignments. Once assignments are submitted, they will show up within the assignment page and you can view their submission replay, and grade assignments. 

# Student view
With a student account, you can enroll into different classrooms and open assignments. You can open a workspace which is linked to the assignment and start working in there. When you are done, submit your assignment. When you get your grades back, it should be viewable in the respective assignment page.
