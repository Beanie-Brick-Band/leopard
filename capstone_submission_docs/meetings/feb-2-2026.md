### **1. Key Discussion Points**

#### **1.1. Grade Management Feature Set**

The team identified critical grade management requirements:

- **Grade Publication:** Teachers can publish grades individually or all at once with confirmation dialog
- **Grade Hiding:** Ability to hide grades after publication for privacy/review purposes
- **Student Grade View:** Students see their grades only after teachers publish them
- **Submission View State:** Teachers can see publish status on submission details
- **Grade Input Validation:** Numeric grades with proper validation and UI improvements (remove spinner arrows)

#### **1.2. Student & Teacher Submission UI**

Two separate submission interfaces required:

  - Next/previous navigation with filtering support
- **Student Interface:**
  - Frozen replay view showing submitted code before final submission
  - Grade display when available
  - Assignment details before enrollment
  - Submit button appears only when viewing assignment details
