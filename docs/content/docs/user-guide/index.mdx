# User Guide

This section is for people using Leopard in a class.

Leopard helps teachers organize coding assignments and helps students complete and submit them in a consistent workspace.

## Start Here

- [Getting Started](/docs/user-guide/getting-started)
- [Students Overview](/docs/user-guide/students)
- [Teachers Overview](/docs/user-guide/teachers)
- [FAQ and Troubleshooting](/docs/user-guide/faq)

## What Leopard Is

Leopard is a classroom coding platform.

In normal day-to-day use:

- teachers create classrooms
- teachers create assignments
- students join classrooms
- students launch workspaces and complete their work
- teachers review, grade, and publish results

## Key Terms

| Term | Meaning |
| --- | --- |
| Classroom | A course or section in Leopard |
| Assignment | A piece of work inside a classroom |
| Workspace | The coding environment for an assignment |
| Submission | The work Leopard records when a student turns something in |
| Replay | A playback view of how code changed over time |
| Grade | The score entered by the teacher |
| Feedback | Written comments from the teacher |
