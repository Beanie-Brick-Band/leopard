## What Teachers Usually Do in Leopard

Most teachers use Leopard to:

- create classrooms
- create assignments
- upload starter code
- monitor submissions
- review replay
- grade student work
- publish grades

## Teacher Workflow

1. Sign in.
2. Create a classroom.
3. Create an assignment.
4. Add instructions and starter code if needed.
5. Wait for students to submit.
6. Review submissions.
7. Grade and publish results.
