# Classrooms

## Teacher Dashboard

The teacher dashboard is the main entry point for classroom management.

It usually shows:

- your classrooms
- assignment counts
- a button to create a classroom

If you have no classrooms yet, Leopard shows an empty state that prompts you to create one.

## Creating a Classroom

To create a classroom:

1. Open the teacher dashboard.
2. Click `Create Classroom`.
3. Enter a classroom name.
4. Click `Create Classroom`.

Good classroom names are easy for students to recognize, such as:

- `Intro to Programming`
- `CS101 Lab A`
- `Data Structures Spring 2026`

## Managing a Classroom

Inside a classroom, you can usually:

- view assignments
- create new assignments
- view enrolled students
- promote or remove assistants
- remove students
- delete the classroom

The classroom page is the main control center for the course.

### Assignment Area

This area usually shows:

- every assignment in the classroom
- due dates
- a quick action to open each assignment

### Student Area

This area usually shows:

- enrolled students
- student names
- email addresses when available
- whether a person is a student or TA

## Managing Students

Teachers can manage the classroom roster from the classroom page.

Typical actions include:

- removing a student from the classroom
- promoting a student to TA
- removing TA status

Use these actions carefully, especially in active classes.

## Tips

- Use clear classroom names.
- Check the roster before posting the first assignment.
- Be careful when removing students from active classes.
