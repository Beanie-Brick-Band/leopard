# Classrooms

Teachers start in `Teacher Dashboard`, create a classroom there, and then manage that classroom from a dedicated page.

## Creating a Classroom

1. Open the teacher dashboard.
2. Click `Create Classroom`.
3. Enter a classroom name.
4. Click `Create Classroom`.

Good classroom names are easy for students to recognize, such as:

- `Intro to Programming`
- `CS101 Lab A`
- `Data Structures Spring 2026`

Once the classroom exists, this page becomes the control center for the course:

- `New Assignment` creates more work inside the class.
- `Assignments` lists each assignment and opens its detail page.
- `Enrolled Students` shows the current roster.
- `Delete Classroom` removes the class when you no longer need it.

## Managing Students

Use the action menu on a student row when you need to:

- promote a student to TA
- remove TA status
- remove a student from the classroom

Use these actions carefully once a class is active, because the roster controls who can access assignments.

## Tips

- Use clear classroom names.
- Check the roster before posting the first assignment.
- Be careful when removing students from active classes.
