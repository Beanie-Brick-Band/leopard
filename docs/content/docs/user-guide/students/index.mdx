## What Students Usually Do in Leopard

Most students use Leopard to:

- join a class
- open assignments
- launch a coding workspace
- complete the work
- submit before the due date
- return later to see grades and feedback

## Student Workflow

1. Sign in.
2. Join the correct class.
3. Open an assignment.
4. Launch the workspace.
5. Complete the work.
6. Submit before the deadline.
7. Check for grades and feedback later.
