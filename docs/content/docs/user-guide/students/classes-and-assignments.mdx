# Classes and Assignments

## Student Dashboard

After signing in, students usually see:

- `Enrolled Classes`
- `Available Classes`

### Enrolled Classes

These are classes you have already joined.

Each class card may show:

- the class name
- a short description
- the number of assignments

### Available Classes

These are classes you can still join.

If your class appears here:

1. Open it.
2. Review the class details.
3. Click `Enroll`.

After that, the class should appear under `Enrolled Classes`.

## Joining a Class

To join a class:

1. Open the class from `Available Classes`.
2. Click `Enroll`.

Once enrolled, you can:

- view assignments
- open assignment details
- launch workspaces

If you do not see your class, ask your teacher whether:

- the class has been created
- you are using the right account
- Leopard access is enabled for your course

## Opening a Class

Inside a class, you will usually see:

- the class name
- a short summary
- your enrollment status
- the list of assignments

Each assignment usually includes:

- the assignment name
- due date
- a submission status badge if you already submitted
- `Launch Workspace`
- `View Details`

## Opening an Assignment

The assignment page is where you do most of your work.

You may see:

- assignment title
- due date
- full assignment description
- your submission status
- the last edited time for your assignment workspace
- buttons for launching and submitting
- grade and feedback, if released

## Reading the Assignment

Always read the assignment description before you start coding.

This page may include:

- the goal of the assignment
- required steps
- important files
- grading notes
- submission expectations

## Student Tips

- Open the assignment page before launching the workspace so you understand the task.
- Double-check the due date early.
- If a class is missing, confirm that you are signed into the correct account.
