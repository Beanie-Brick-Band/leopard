# Classes and Assignments

The student flow starts on `My Classes`, then moves into a classroom page with one card per assignment.

1. Sign in and open `My Classes`.
2. Look under `Enrolled Classes` for classes you already joined.
3. Look under `Available Classes` for classes you can still inspect or join.
4. If a class is available but not enrolled yet, open it and click `Enroll` from the classroom page.

## Joining a Class

If Leopard shows the class in `Available Classes` but not `Enrolled Classes`, use this flow:

1. Click `View Class`.
2. Review the class page.
3. Click `Enroll`.
4. Return to `My Classes` if you want to confirm the class now appears in `Enrolled Classes`.

Each assignment card includes the information students usually need first:

- assignment name
- due date
- a `Submitted` badge when Leopard already has your submission
- `Launch Workspace`
- `View Details`

Use `View Details` before you start coding whenever you want to read the prompt carefully.

## If a Class Is Missing

- Confirm you are signed into the correct Leopard account.
- Ask your teacher whether the classroom has already been created.
- Ask whether your course expects students to self-enroll or whether enrollment is managed for you.

## Student Tips

- Open the assignment details page before launching the workspace.
- Check the due date early instead of waiting until submission time.
- If you are enrolled in several classes, use the classroom page to confirm you are working in the correct one.
