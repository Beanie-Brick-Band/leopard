# Getting Started

This page is the best place to begin if you are new to Leopard.

## Signing In

From the Leopard homepage, you will usually see:

- `Log In`
- `Sign Up`

If you already have an account:

1. Click `Log In`.
2. Enter your credentials.
3. Open the app.

If you are new:

1. Click `Sign Up`.
2. Create your account.
3. Sign in.

## After You Sign In

Leopard sends you to the experience that matches your account.

- Students are taken into the student area.
- Teachers are shown the teacher dashboard.

If you expected teacher tools but do not see them, your account may not have teacher access yet.

## The Main Parts of Leopard

Most people using Leopard will see these main ideas:

| Term | Meaning |
| --- | --- |
| Classroom | A course or section in Leopard |
| Assignment | A piece of work inside a classroom |
| Workspace | The coding environment for an assignment |
| Submission | The record created when a student turns something in |
| Replay | A playback view of how code changed over time |
| Grade | The score entered by the teacher |
| Feedback | Written comments from the teacher |
