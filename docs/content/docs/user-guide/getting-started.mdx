# Getting Started

Start here if this is your first time in Leopard. The current web app uses a single sign-in page and then routes students and teachers to different dashboards automatically.

1. Click `Log In`.
2. Enter the email address tied to your Leopard account.
3. Enter your password.
4. Click `Sign in`.

If you do not have an account yet, use `Sign Up` from the same screen before returning to the sign-in form.

## After You Sign In

- Students are taken into the student area.
- Teachers are shown the teacher dashboard.
- Leopard decides this from your account role, so you do not need to pick a mode manually.

If you expected teacher tools but land in the student flow instead, your account probably does not have teacher access yet.

## The Main Parts of Leopard

You will see these terms throughout the rest of the guide:

| Term | Meaning |
| --- | --- |
| Classroom | A course or section in Leopard |
| Assignment | A piece of work inside a classroom |
| Workspace | The coding environment for an assignment |
| Submission | The record created when a student turns something in |
| Replay | A playback view of how code changed over time |
| Grade | The score entered by the teacher |
| Feedback | Written comments from the teacher |
